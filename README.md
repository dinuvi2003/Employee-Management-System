# Employee Management System

This is a full-stack Employee Management System developed as part of an academic assignment.

## Project Structure
- backend/  → Spring Boot REST API
- frontend/ → React frontend

## Backend Tech Stack
- Spring Boot
- Java
- MySQL

## Frontend Tech Stack
- React
- Vite
- Axios

## How to Run Backend
1. Open backend folder in IntelliJ IDEA
2. Configure `application.properties`
3. Run `DemoApplication.java`

## How to Run Frontend
1. cd frontend
2. npm install
3. npm run dev
